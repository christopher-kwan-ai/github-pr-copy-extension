# GitHub PR Copy for Slack

A Chrome extension that adds a convenient copy button to GitHub pull request pages, allowing you to quickly copy PR links formatted for Slack.

## Features

- Adds a small copy button next to the PR number in the page header
- Copies PR title, number, and URL in a format optimized for Slack
- Creates clickable links when pasted into Slack
- Matches GitHub's UI styling with dark mode support
- Lightweight and fast

## Installation

### Method 1: Load Unpacked Extension (Development)

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** by toggling the switch in the top-right corner
3. Click **Load unpacked**
4. Navigate to and select the `github-pr-copy-extension` folder
5. The extension should now appear in your extensions list

### Method 2: Install from ZIP

1. Extract the `github-pr-copy-extension.zip` file
2. Follow the steps above for Method 1 using the extracted folder

## Usage

1. Navigate to any GitHub pull request page
2. Look for the copy button (ðŸ“‹ icon) next to the PR number in the header
3. Click the button to copy the PR link
4. Paste into Slack - it will appear as a clickable link with the format: `[PR Title] #[Number]`

## Updating the Extension

If you make changes to the extension code:

1. Go to `chrome://extensions/`
2. Find the "GitHub PR Copy for Slack" extension
3. Click the refresh/reload icon (ðŸ”„)
4. Reload any open GitHub PR pages
